About deployKF

deployKF builds machine learning platforms on Kubernetes. We combine the best of {% include ".icons/custom/kubeflow-color.svg" %} Kubeflow , {% include ".icons/custom/airflow-color.svg" %} Airflow† , and {% include ".icons/custom/mlflow-color.svg" %} MLflow† into a complete platform that is easy to deploy and maintain.
†Coming soon, see our current and future tools.

Key Features

deployKF provides centralized configs, seamless in-place upgrades, and the flexibility to run on any Kubernetes cluster, including self-hosted and cloud services.
Our goal is that any Kubernetes user can build a machine learning platform for their organization, without needing specialized MLOps knowledge, or a team of experts to maintain it.
